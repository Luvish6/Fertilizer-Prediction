# Fertilizer Prediction

This project predicts the appropriate fertilizer based on soil and crop parameters using machine learning.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Description
The model analyzes soil nutrients and crop requirements to recommend suitable fertilizers, helping improve yield and reduce misuse.

## How to Run
1. Clone the repository
2. Install required libraries
3. Run the notebook or Python script
